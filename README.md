# Zainab Gull Khan — Interactive Portfolio

A modern, editorial-inspired portfolio built with **Next.js 15**, **React 19**, **TypeScript**, and **MongoDB**. Designed to go beyond the traditional developer portfolio, this project combines immersive storytelling, handcrafted aesthetics, and interactive web experiences to showcase my academic journey, professional experience, and software engineering projects.

## Features

- Interactive editorial-inspired UI/UX
- Fully responsive design
- TypeScript-based full-stack architecture
- Next.js App Router
- MongoDB integration with Mongoose
- Secure API routes
- Smooth animations using Framer Motion, GSAP, and Lenis
- Contact form with backend validation
- SEO and performance optimized

## Tech Stack

- **Frontend:** Next.js, React, TypeScript, Tailwind CSS
- **Animations:** Framer Motion, GSAP, Lenis
- **Backend:** Next.js Route Handlers, TypeScript
- **Database:** MongoDB, Mongoose
- **Validation:** Zod
